# Git提交说明

## 方法一：使用批处理脚本（推荐）

1. 双击运行 `git_push.bat` 文件
2. 等待脚本执行完成
3. 查看GitHub仓库确认提交成功

---

## 方法二：手动执行命令

打开命令行（CMD或PowerShell），进入项目目录，依次执行：

```bash
# 1. 初始化Git仓库
git init

# 2. 添加远程仓库
git remote add origin https://github.com/hoshinoai12345678/AI-Health-Assistant.git

# 3. 添加所有文件
git add .

# 4. 提交
git commit -m "Initial commit: AI健康助手完整项目"

# 5. 推送到GitHub
git branch -M main
git push -u origin main --force
```

---

## 注意事项

1. **首次推送**: 如果仓库已存在内容，使用 `--force` 会覆盖远程仓库
2. **认证**: 可能需要输入GitHub用户名和密码（或Personal Access Token）
3. **大文件**: 如果有大文件，可能需要使用Git LFS

---

## 如果遇到认证问题

### 使用Personal Access Token

1. 访问 GitHub Settings → Developer settings → Personal access tokens
2. 生成新的token（勾选repo权限）
3. 使用token作为密码进行推送

### 或者使用SSH

```bash
# 添加SSH远程地址
git remote set-url origin git@github.com:hoshinoai12345678/AI-Health-Assistant.git
```
