# AI健康助手 - 快速开始指南

---

## 🎯 5步快速启动

```
1️⃣ 申请账号（30分钟）
   ├─ 微信小程序 → AppID + AppSecret
   └─ 阿里云通义千问 → API Key

2️⃣ 安装软件（20分钟）
   ├─ Docker Desktop
   └─ 微信开发者工具

3️⃣ 配置项目（10分钟）
   ├─ 复制 .env.example 为 .env
   ├─ 填入AppID、AppSecret、API Key
   └─ 修改小程序AppID（2个地方）

4️⃣ 启动服务（10分钟）
   ├─ docker-compose up -d
   ├─ 初始化数据库
   └─ 验证服务

5️⃣ 开发测试（30分钟）
   ├─ 导入小程序项目
   └─ 真机调试
```

---

## 第一步：申请账号

### 微信小程序
- 网址：https://mp.weixin.qq.com/
- 获取：**AppID** 和 **AppSecret**
- 详细步骤：查看 `微信小程序开发部署指南.md`

### 阿里云通义千问
- 网址：https://dashscope.aliyun.com/
- 获取：**API Key**（免费100万tokens）
- 详细步骤：查看 `AI模型API申请配置指南.md`

---

## 第二步：安装软件

### Docker Desktop
- 下载：https://www.docker.com/products/docker-desktop
- 安装后重启电脑

### 微信开发者工具
- 下载：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

---

## 第三步：配置项目

### 1. 克隆项目
```bash
git clone https://github.com/hoshinoai12345678/AI-Health-Assistant.git
cd AI-Health-Assistant
```

### 2. 配置后端
```bash
# 复制环境变量文件
cp .env.example .env

# 编辑.env文件，填入你的配置
notepad .env  # Windows
nano .env     # Mac/Linux
```

**必须修改的配置**：
```env
WECHAT_APP_ID=你的AppID
WECHAT_APP_SECRET=你的AppSecret
TONGYI_API_KEY=你的API_Key
```

### 3. 配置小程序
**文件1**：`miniprogram/project.config.json`
```json
"appid": "你的AppID"
```

**文件2**：`miniprogram/utils/config.ts`
```typescript
APP_ID: '你的AppID'
```

---

## 第四步：启动服务

```bash
# 启动Docker服务
docker-compose up -d

# 初始化数据库
docker-compose exec backend alembic upgrade head
docker-compose exec backend python scripts/import_resources.py

# 验证服务
# 访问：http://localhost:8000/health
# 访问：http://localhost:8000/docs
```

---

## 第五步：开发小程序

1. 打开微信开发者工具
2. 导入项目 → 选择 `miniprogram` 目录
3. 输入你的AppID
4. 详情 → 本地设置 → 勾选"不校验合法域名"
5. 点击"预览"进行真机调试
