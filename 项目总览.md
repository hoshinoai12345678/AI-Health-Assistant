## 🚀 如何启动

### 快速启动（本地开发）

```bash
# 1. 启动后端
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

# 2. 启动小程序
# 使用微信开发者工具打开miniprogram目录
```

### Docker启动（推荐）

```bash
cd docker
docker-compose up -d
```

详见：[部署指南](docs/deployment.md)
